####  HOW TO RUN  ######

git clone https://github.com/alandefreitas/matplotplusplus.git

sudo apt install gnuplot

cmake .

make 

./program



####### Dependencies ##########

- cmake version 3.14+ installed
- make sure your C++ compiler supports C++17

########  Tested  #############

- ubuntu 20.04
- g++ 9.3.0
- cmake 3.16.0
- gnuplot 5.2 patchlevel 8
